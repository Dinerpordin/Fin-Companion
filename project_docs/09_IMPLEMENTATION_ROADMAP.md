# Implementation Roadmap

## Development context

The project is being built in a lean setup with strong existing assets:
- production-ready loan calculation engine
- near-complete accounting tool
- comparison tool foundation
- active AI-assisted development workflow

## Work streams

### Stream 1: Core tools integration
- loan engine
- comparison engine
- accounting integration
- assistant routing into tools

### Stream 2: Product and UX
- user journeys
- Bangla copy
- wireframes
- interface polish
- low-literacy flows

### Stream 3: Data and infrastructure
- scraping
- schemas
- databases
- APIs
- refresh pipelines
- analytics foundation

### Stream 4: Validation and partnerships
- borrower interviews
- agent and shopkeeper interviews
- NGO/MFI outreach
- regulatory preparation

## Build phases

### Phase 0: Integration foundation
- unify navigation across existing tools
- clean current demo
- standardize Bangla copy
- add visible privacy statement
- finalize schemas and contracts

### Phase 1A: Foundation
- Loan Reality Checker single and multi-loan flow
- comparison for FD, DPS, and personal loans
- assistant text MVP
- privacy framework and trust messaging
- anonymous binned analytics
- initial interviews and user validation

### Phase 1B: Depth
- expand DPS and savings coverage
- Financial Health Assessment live
- accounting tool live in simplified form
- What-If Planner live
- full Bangla UX polish
- locator
- assistant voice input/output
- district-level pilot

### Phase 1C: Scale
- USSD/SMS-lite fallback
- document pack generator
- research warehouse
- insurance informational comparison
- SEO and content programme
- stronger investor and partner demo

### Phase 2A: Regulatory and partnerships
- Open Banking engagement
- BFIU preparation if application flows expand
- IDRA engagement for insurance aggregation
- full PDPO compliance audit
- partnership agreements with banks and MFIs

### Phase 2B: Monetisation
- clearly labelled application commissions
- clearly labelled featured listings
- white-label APIs
- anonymised statistical products
- freemium document packs
- insurance commissions when permitted
