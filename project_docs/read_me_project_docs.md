## Product position

A Bangla-first, privacy-focused, mobile-first financial companion for Bangladesh that helps users:

- understand existing and upcoming loan decisions,
- compare better formal alternatives,
- assess financial health,
- track income and expenses privately,
- find the nearest suitable provider,
- understand how to apply,
- interact through text and voice without receiving personalised financial advice.
